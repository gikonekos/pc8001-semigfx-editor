01

Overview

PC-8001 Semi-Graphic Editor is a browser-based drawing tool for creating semi-graphic patterns for the NEC PC-8001 home computer (1979). It operates entirely within a single HTML file — no installation or server is required.

The editor produces Z80 assembly DB data and PNG images that faithfully represent the PC-8001's semi-graphic character set, including color attribute data.

ℹ SYSTEM REQUIREMENTS

Any modern browser (Chrome / Firefox / Safari / Edge latest). No internet connection required after loading. Works fully offline.

02

Screen Specification

The PC-8001 semi-graphic mode uses character codes in the range 0xA0–0xBF. Each character cell is divided into a 2×3 grid of sub-pixels (6 bits total).

CHARACTER CELL BIT LAYOUT
BitPositionSub-pixel
bit 0row 0, col 0Top-left
bit 1row 0, col 1Top-right
bit 2row 1, col 0Mid-left
bit 3row 1, col 1Mid-right
bit 4row 2, col 0Bottom-left
bit 5row 2, col 1Bottom-right

Character code = 0xA0 | bits. An empty cell (all sub-pixels off) is 0xA0; a fully lit cell is 0xBF.

COLOR PALETTE
0 BLACK
1 RED
2 GREEN
3 YELLOW
4 BLUE
5 MAGENTA
6 CYAN
7 WHITE
COLOR ATTRIBUTE CONSTRAINT

On real PC-8001 hardware, color attribute changes are limited to 20 per row. The editor displays a warning in the ASM Preview panel when this limit is exceeded, but does not enforce it.

⚠ HARDWARE CONSTRAINT

Exceeding 20 color changes per row will cause display corruption on real PC-8001 hardware. Verify on real hardware or an emulator before use.

03

Tools

Select tools from the left toolbar or via keyboard shortcuts.

✏️
Pen P
Left-click or drag to turn sub-pixels ON. The selected color is applied to the entire character cell.
🧹
Erase E
Left-click or drag to turn sub-pixels OFF, one at a time.
🪣
Fill F
Flood-fill connected cells sharing the same bit pattern and color with the current color (fully filled).
▤
Select S
Drag to define a rectangular selection (character-cell units). Used for partial ASM / PNG export.
RIGHT-CLICK ERASE

Right-clicking on the canvas erases the sub-pixel under the cursor, regardless of the active tool. Right-click drag performs continuous erase. If the browser context menu appears, right-click erase is not available in that environment.

04

Grid Size

The canvas size is specified in character cells (W × H). The actual dot resolution is shown in real time as you type: dots = W×2 × H×3.

ParameterRangeDefault
W (cols)1 – 8080
H (rows)1 – 2525

Press ■ Apply to apply the new size. Existing content is preserved as far as possible; cells outside the new bounds are discarded.

◆ NOTE

The default 80×25 character grid produces 160×75 sub-pixel dots, matching the PC-8001's standard text screen width.

05

Image Import

Click ⬆ Load Image in the left toolbar to import a PNG, JPEG, or other browser-supported image. The image is automatically converted to the current canvas size and PC-8001 constraints.

CONVERSION PROCESS

The image is scaled to the canvas's sub-pixel resolution (COLS×2 × ROWS×3 pixels). Then, for each character cell:

  • Bit pattern — Each of the 6 sub-pixels is compared against a luminance threshold (half the cell's average luminance + 20). Sub-pixels brighter than the threshold are set ON.
  • Color attribute — Each sub-pixel's RGB value is mapped to the nearest PC-8001 color. The most common non-black color in the cell is used as the cell's color attribute.
◆ TIP

Image import supports Undo (Ctrl+Z). The imported result can be refined with the Pen and Erase tools.

06

ASM Preview

The right panel shows a live ASM preview that follows the mouse cursor. The displayed region is always centered on the current cursor position and fills the panel as much as possible.

ElementDescription
[A0H]The character cell currently under the cursor (highlighted with brackets).

ℹ NOTE

An empty cell (no sub-pixels lit) always appears as A0H.

For ASM export, a modal dialog shows the generated code. Use Copy to copy to clipboard and simultaneously download the .asm file. PNG export outputs at native sub-pixel resolution (1 pixel per sub-pixel, no zoom applied).

To export a selection, first define a region with the ▤ Select tool, then use the Export (Selection) buttons.

⚠ BROWSER SECURITY RESTRICTION

Due to browser security restrictions, exported files are always saved to the browser's default download folder. The save destination cannot be chosen. This applies to both ASM (.asm) and PNG files.

08

ASM Format

The exported .asm file contains two separate data blocks in Z80 DB format, row-major order:

; PC-8001 Semi-Graphic Data
; Grid: 10 cols x 3 rows (chars)
; Color attr: 0=BLK 1=RED 2=GRN 3=YEL 4=BLU 5=MAG 6=CYN 7=WHT

; -- Pattern data (char codes) --
  DB  0A0H,0BFH,0B4H,...  ; row 0
  DB  0AAH,0A0H,0BCH,...  ; row 1

; -- Color attribute data --
  DB  7H,2H,1H,...        ; row 0
  DB  7H,7H,4H,...        ; row 1

Character codes range from 0A0H (blank) to 0BFH (fully filled). Color attribute values are 0–7 corresponding to the 8-color palette.

01

概要

PC-8001 セミグラフィックエディタは、NEC PC-8001（1979年）向けのセミグラフィックパターンを作成するブラウザベースの描画ツールです。単一のHTMLファイルで動作し、インストールやサーバーは不要です。

Z80アセンブリ形式のDBデータおよびPNG画像を出力します。カラーアトリビュートデータも含みます。

ℹ 動作環境

モダンブラウザ（Chrome / Firefox / Safari / Edge 最新版）。インターネット接続不要。HTMLファイルをブラウザで開くだけで動作します。

02

画面仕様

PC-8001のセミグラフィックモードは、0xA0〜0xBFの範囲の文字コードを使用します。各文字セルは2×3のサブピクセル（6ビット）に分割されています。

文字セルのビット配置
ビット位置サブピクセル
bit 0行0, 列0左上
bit 1行0, 列1右上
bit 2行1, 列0左中
bit 3行1, 列1右中
bit 4行2, 列0左下
bit 5行2, 列1右下

文字コード = 0xA0 | bits。空セル（全消灯）は0xA0、全点灯は0xBF。

カラーパレット
0 黒
1 赤
2 緑
3 黄
4 青
5 マゼンタ
6 シアン
7 白
カラーアトリビュート制約

実機では、1行あたりの色変更回数は最大20回に制限されています。エディタはこの制限を超えた場合にASMプレビューに警告を表示しますが、強制はしません。

⚠ ハードウェア制約

実機で1行あたり20回を超える色変更を行うと、画面表示が乱れます。実機またはエミュレータで最終確認することを推奨します。

03

ツール

左ツールバーまたはキーボードショートカットで選択します。

✏️
Pen P
左クリック・ドラッグでサブピクセルをONにします。選択中の色が文字セル全体に適用されます。
🧹
Erase E
左クリック・ドラッグでサブピクセルを1つずつOFFにします。
🪣
Fill F
同じビットパターン・色を持つ隣接セルを現在の色で塗りつぶします（全点灯）。
▤
Select S
ドラッグで矩形選択範囲を指定します（文字セル単位）。選択範囲のみのASM/PNGエクスポートに使用します。
右クリック消去

キャンバス上で右クリックすると、アクティブなツールに関わらずサブピクセルを消去します。右クリックドラッグで連続消去が可能です。ブラウザのコンテキストメニューが表示される環境では右クリック消去は使用できません。

04

グリッドサイズ

キャンバスサイズは文字セル単位（W × H）で指定します。実際のドット解像度は入力中にリアルタイムで表示されます（ドット数 = W×2 × H×3）。

パラメータ範囲デフォルト
W（列数）1 – 8080
H（行数）1 – 2525

■ Apply ボタンを押すと新しいサイズが適用されます。既存の描画内容は可能な限り保持されます。

◆ 備考

デフォルトの80×25文字グリッドは160×75サブピクセルで、PC-8001の標準テキスト画面幅に対応しています。

05

画像読み込み

左ツールバーの⬆ Load Imageボタンをクリックして、PNG、JPEGなどのブラウザ対応画像ファイルを読み込みます。現在のキャンバスサイズとPC-8001の制約に自動変換されます。

変換処理

画像をキャンバスのサブピクセル解像度（COLS×2 × ROWS×3ピクセル）にリサイズした後、各文字セルで以下を行います：

  • ビットパターン — 6つのサブピクセルそれぞれの輝度を閾値（セル内平均輝度の半分＋20）と比較し、明るい場合はONにします。
  • カラーアトリビュート — 各サブピクセルのRGB値をPC-8001の8色に最近傍マッピングし、セル内で最も多い非黒色をカラーアトリビュートとして採用します。
◆ ヒント

画像読み込みはUndo（Ctrl+Z）に対応しています。読み込み後はPen・Eraseツールで調整できます。

06

ASMプレビュー

右パネルにはマウスカーソルに追従するリアルタイムASMプレビューが表示されます。表示範囲は常にカーソル位置を中心に、パネルの高さ・幅の限界まで表示します。

要素説明
[A0H]現在のカーソル位置の文字セル（括弧でハイライト表示）。
c23-30 r0-5プレビューの表示列・行範囲。
⚠ COLOR>…警告：カラーアトリビュート変更回数が20回を超えている行があります。
Patternセクション文字コード（A0H〜BFH）を行優先順で表示。
Colorセクション各セルのカラーアトリビュートインデックス（0〜7）を同順で表示。
ℹ 備考

空セル（サブピクセルがすべてOFF）は常にA0Hで表示されます。

07

エクスポート

ボタン出力範囲
⬇ ASM (Z80 DB)テキストファイル (.asm)キャンバス全体
⬇ PNGPNG画像キャンバス全体
⬇ ASM — selectionテキストファイル (.asm)選択範囲のみ

ASMエクスポートはモーダルダイアログに生成コードを表示します。Copyボタンでクリップボードにコピーし、同時に.asmファイルをダウンロードします。PNGエクスポートはサブピクセル等倍（ズームなし）の解像度で出力します。

選択範囲エクスポートを使用するには、先に▤ Selectツールで範囲を指定してください。

⚠ ブラウザのセキュリティ制約

ブラウザのセキュリティ制約により、エクスポートファイルは常にブラウザのデフォルトダウンロードフォルダに保存されます。保存先を選択することはできません。ASM（.asm）およびPNGファイルのいずれも同様です。

08

出力フォーマット

出力される.asmファイルには、Z80 DB形式で行優先順の2つのデータブロックが含まれます：

; PC-8001 Semi-Graphic Data
; Grid: 10 cols x 3 rows (chars)
; Color attr: 0=BLK 1=RED 2=GRN 3=YEL 4=BLU 5=MAG 6=CYN 7=WHT

; -- Pattern data (char codes) --
  DB  0A0H,0BFH,0B4H,...  ; row 0
  DB  0AAH,0A0H,0BCH,...  ; row 1

; -- Color attribute data --
  DB  7H,2H,1H,...        ; row 0
  DB  7H,7H,4H,...        ; row 1

文字コードは0A0H（空白）〜0BFH（全点灯）の範囲です。カラーアトリビュートは8色パレットに対応する0〜7の値です。
